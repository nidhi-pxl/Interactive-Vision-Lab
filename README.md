# Interactive Vision Lab

A collection of interactive computer vision and graphics projects built entirely in Python.

## Overview

Interactive Vision Lab is a modular repository demonstrating real-time computer vision, gesture recognition, particle systems, and interactive graphics.

Each demo builds upon previous components while keeping the architecture modular and reusable.

## Demos

1. Hand Tracking
2. Finger Counter
3. Pinch Detection
4. Hand Painter
5. Particle Trails
6. Blooming Flowers
7. Face Mesh
8. Face Particles
9. Fruit Ninja

## Technologies

- Python
- OpenCV
- MediaPipe
- ModernGL
- Pygame
- NumPy

## Repository Structure

- `src/` — reusable modules
- `demos/` — independent demo applications
- `assets/` — images, shaders, particles, sounds
- `docs/` — implementation notes and research
- `screenshots/` — demo screenshots
- `videos/` — demo recordings (future)

## Goals

- Learn modern computer vision
- Learn gesture recognition
- Build reusable vision components
- Explore particle systems
- Build interactive graphics
- Create portfolio-quality demonstrations
